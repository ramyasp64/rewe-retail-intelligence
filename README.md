## 🎯 Project Overview

This BI project explores a real-world retail dataset simulating REWE Group's sales operations across product categories (Fresh, Organic, Regional), regions, competitors, and loyalty program segments. Visual insights were created in Power BI and packaged as a professional slide deck, surfacing **four key strategic findings** with actionable recommendations.

---

## 💡 Key Findings & Recommendations

| Finding | Strategic Recommendation |
|---------|--------------------------|
| **Loyalty members generate 1,953 EUR vs. 1,809 EUR avg. sales (+8%)** | Expand loyalty program to underserved regions, measurable revenue uplift |
| **Munich and Berlin lead in Organic product sales** | Focus regional marketing and loyalty campaigns in these cities |
| **Lidl offers the lowest average product prices** | Introduce price-matching strategies in high-competition areas |
| **Sales trends vary seasonally, Organic peaks March to June** | Align supply chain and inventory with seasonal demand patterns |

## 🛠️ Technical Implementation

**Data Processing (Power BI + Excel)**
- Imported and cleaned the retail Excel dataset
- Normalised product categories and regional names
- Converted string dates to DateTime format
- Created DAX calculated columns: Average Sales, Sales Uplift with Loyalty Program, Monthly Trends

**Power BI Features Used**
- Page-level and visual-level slicers (Category, Region filtering)
- Diverging gradient colour scales for geographic heatmaps
- Tooltip configuration for drilldown analytics
- Combo charts (dual-axis sales & price comparison)
- Filled maps for regional geospatial visualisation
